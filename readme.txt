== Description ==
This plugin is used to integrate Sadad QA Payment gateway with Woocommerce.

Pay Your Order with Sadad is the simplest of the available ways to do payment integration to your website. 

The person who clicks the Pay via Sadad button will be taken to Pay with Sadad website where user will complete the payment using Credit Card, NAPS card, Sadad, Apple Pay or Google Pay and Pay with Sadad returns back the browser handle to the website after payment confirmation. Website will also receive a notification, which will be a server-to-server call as soon as the payment is done.

== Installation ==
Installation

1. Login your WordPress admin and navigate to Plugins->Add New 

2. Find the Sadad QA Payment plugin and activate the module.

3. After the successful activation of module, Visit the WooCommerce => settings page, and click on the Payments tab.

4. Click On Checkout and display "Pay With Sadad" payment gateway.

5. Enable the Payment Method, and Set all details and  Click Save.(See screenshot)

Now you can see *pay with Sadad* payment option on checkout page.

For Create Merchant ID Go to <a href="https://panel.sadad.qa/" target="_blank">Sadad Merchant Website</a>.
